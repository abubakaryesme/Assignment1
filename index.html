COMMAND

1. git init

The command "git init" is to intialize a new local GitHub repository. It creates the .git repository folder.

2. git status

The "git status" commmand is used to find out all untracked files. It will display the all the local files which are modified and not commited.

3. git add

The "git add" commmand is used to add all the untracked files to a staging area. All the new, modified and deleted files (untracked files) are added to git staging area.

4. git commit

The "git commit" commmand is used for all the files which are on staging area and are not stored in local repository. It will save all the changes which are on stating area to local repository.

5. GitHub Repository

To create a new GitHub repository, you have to give owner and repository name as well. Repository can be either public or private. By clicking on "Create repository", a GitHub Repository will be created.

6. git remote add

The "git remote add" command add a new remote of repository. it takes two arguments. One is a remote name and other is remote URL.

7. git clone

The "git clone" command is used to point to an existing repository and make a clone of that repository at destination location.

8. git push

The "git push" command is used to upload all the local repository content (the content which are commited in local repository) to a remote repository.
